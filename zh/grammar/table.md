## 表格

### 基本用法
1. 表格使用`|`和`-`来绘制
2. 表格的对齐用`:`来标识

示例：
```
| Title   | Desc                                          |
| :------ | :-------------------------------------------: |
| Name    | HwangJR                                       |
| Email   | [HwangJR](mailto:huangjianrong2010@gmail.com) |
```

效果：

| Title   | Desc                                          |
| :------ | :-------------------------------------------: |
| Name    | HwangJR                                       |
| Email   | [HwangJR](mailto:huangjianrong2010@gmail.com) |
